How FlashScreener handles and protects your data

This Privacy Policy describes how the FlashScreener Chrome Extension ("we", "our", "the Extension") handles user data.

1. Information Collection

The Extension does not collect, store, transmit, or share any personally identifiable information (PII). All operations occur locally within the user’s browser. We do not use external servers or databases to process or store data.

2. Permissions

The Extension may request certain Chrome permissions (such as context menus, storage, or activeTab) in order to provide its core functionality. These permissions are used exclusively for the features described in the Extension’s description and are not used to access personal or sensitive data.

3. Data Sharing

We do not share, sell, or transfer user data to third parties under any circumstances.

4. Extension Third-Party Services

The Extension does not use third-party analytics, advertising networks, or tracking tools.

5. User Control

Since no personal data is collected, there is no data for the user to request access to, modify, or delete. The user can uninstall the Extension at any time through their browser settings.

6. Updates

We may update this Privacy Policy from time to time. Any updates will be published at this URL with a new effective date.
